## 🔁 Engagement Loops to Push

What loops are we trying to create?

- Builders → post updates → get feedback
- Community → engages → builders stay motivated
- Feedback givers → rewarded → more participation

Be explicit—this is where momentum comes from.

---

## 📊 Success Signals

How do we know the week worked?

- [ ] Builders posting updates
- [ ] GitHub activity increasing
- [ ] Feedback sessions happening
- [ ] Cross-engagement (comments, replies)
- [ ] Visible momentum (people talking about projects)

---

## 🧠 Risks / Failure Modes

What could go wrong this week?

- Builders stall / don’t start
- Low visibility (people building quietly)
- No feedback requests
- Overthinking / perfectionism

---

## 🛠️ Interventions (if needed)

What will admins do if things stall?

- Directly nudge builders
- Highlight early movers
- Ask specific people for feedback
- Post examples/templates
