# Examples

This folder links to and documents real implementations of the Piece of Pie Playbook.

## Reference Implementation

The original event this playbook was built from:

**Gimbalabs Piece of Pie Hackathon 2026**

- Repo: https://github.com/gimbalabs/Piece-of-Pie-Hackathon
- Website: https://gimbalabs.com/piece-of-pie
- Community: Gimbalabs
- Duration: 12 weeks
- Prize Pool: 33,000 ADA across four tracks
- Ecosystem Track Sponsor: Cardano Foundation

This is the event-specific implementation that was generalized into this playbook.

---

## How to Add Your Event

If you run an event using this playbook, add it here:

```markdown
## {{event_name}} — {{season_name}}

- Repo: [link]
- Website: [link]
- Community: [name]
- Duration: X weeks
- Prize Pool: X [currency]
- Notes: [anything useful for others]
```

Opening a pull request to this file is a great way to:

- contribute back to the playbook
- make your event discoverable to other communities
- share what you learned
