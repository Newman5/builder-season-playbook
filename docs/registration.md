# Registration

## Registration Window

Registration is open during the enrollment period only.

- {{enrollment_start}} to {{enrollment_end}}

No new entries are accepted after the enrollment period closes.

> Registration is not required for the Feedback Pie track. Find more info in the [Feedback Rules](rules/feedback-rules.md).

## One Registration Per Project

Each project should submit one registration.

## What Participants Should Prepare

Before registering, participants should have:

- a project title
- a complete list of all contributors
- one official public repository
- a repository that starts empty for the event
- access to {{community_name}}'s Discord

## Team Registration Rules

- solo projects are allowed
- team projects are allowed
- every contributor must be named during registration
- participants may only join one project

## Official Links

- Registration form: {{registration_form_url}}
- Discord: {{discord_url}}
- Feedback forum: {{discord_feedback_channel_url}}
- Discussion channel: {{discord_discussion_channel_url}}

## Discord Access

Participants should join the {{community_name}} Discord and complete server verification.

After verification:

- use the feedback forum for feedback requests and feedback coordination
- use the discussion channel for general event questions and discussion

## Public Record Expectation

Registration should match the public record that participants maintain throughout the event, including:

- the official repository
- weekly progress updates
- final presentation materials
